# 📚 Proyecto de Promoción de la Lectura

Aplicación web desarrollada para apoyar un proyecto de promoción de la lectura y alfabetización destinado a niños de un barrio en situación de vulnerabilidad, impulsado por Rotaract Club Paraná.

---

## 🎯 Objetivo

El proyecto busca fomentar el hábito de la lectura mediante un sistema de gamificación, donde los niños obtienen puntos y recompensas a medida que leen libros y participan de las actividades propuestas.

---

## 💡 Motivación

Durante las actividades realizadas en el barrio identificamos la necesidad de contar con una herramienta que complementara el trabajo presencial y ayudara a incentivar la lectura de forma más atractiva.

A partir de esa necesidad desarrollé esta aplicación para facilitar el seguimiento de las lecturas y apoyar el proyecto educativo.

---
